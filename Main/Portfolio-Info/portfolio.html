Portfolio Beschrijving

Dit portfolio is ontworpen als een website waarop ik mij als Software Developer kan presenteren en waar ik mijn vaardigheden en projecten kan laten zien. Ook biedt deze website mij de mogelijkheid om te laten zien wat ik tijdens mijn opleiding heb geleerd, en kan ik het in de toekomst gebruiken als professioneel platform voor een stage of een sollicitatie.

De website bevat pagina’s zoals een overzicht, vaardigheden, projecten en contact. Ik heb verschillende portfolio’s bekeken en wireframes gemaakt om het ontwerp te ontwikkelen op basis van onderdelen die ik visueel aantrekkelijk vond.

Het resultaat is een strak ontwerp met moderne en neutrale kleuren, die de website overzichtelijk en professioneel maken. Tijdens het ontwerpen kwamen er nieuwe ideeën in mij op, die ik meteen in de sitemap heb verwerkt. Het doel van deze website is om mijn persoonlijke en professionele ontwikkeling te tonen.

Kleurenschema

Op mijn portfolio website gebruik ik drie kleuren: zwart voor de achtergrond, wit voor de tekst en blauw voor knoppen en belangrijke dingen. De zwarte achtergrond ziet er modern uit. Witte tekst is makkelijk te lezen. Blauw valt op en gebruik ik voor dingen die belangrijk zijn.
